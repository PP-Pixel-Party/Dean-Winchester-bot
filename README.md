### Variables

Read this before you start messing up with your edits

##### Required Variables

‚Ä¢ `ADMINS_ID` : Username or ID of Admin. Separate multiple Admins by space

‚Ä¢ `API_ID` : Get this value from [telegram.org](https://my.telegram.org/apps)

‚Ä¢ `API_HASH` : Get this value from [telegram.org](https://my.telegram.org/apps)

‚Ä¢ `BOT_TOKEN` : Create a bot using [@BotFather](https://telegram.dog/BotFather), and get the Telegram API token

‚Ä¢ `CHANNELS` : Username or ID of channel or group. Separate multiple IDs by space

‚Ä¢ `COLLECTION_NAME` : Name of the collections. Defaults to Telegram_files. If you are using the same database, then use different collection name for each bot

‚Ä¢ `DATABASE_NAME` : Name of the database in [mongoDB](https://www.mongodb.com/). For more help watch this [video](https://youtu.be/mD9veNL7KoE)

‚Ä¢ `DATABASE_URL` : [mongoDB](https://www.mongodb.com/) URI. Get this value from [mongoDB](https://www.mongodb.com/). For more help watch this [video](https://youtu.be/mD9veNL7KoE)

‚Ä¢ `FORCE_SUB` : ID of channel. Without subscribing this channel users cannot use Autofilter

‚Ä¢ `GITHUB_USERNAME` Enter Your Github Username `Pr0fess0r-99`

‚Ä¢ `LOG_CHANNEL` : A channel to log the activities of bot. Make sure bot is an admin in the channel

‚Ä¢ `REPO_USERNAME` : Enter Repository name `Midukki-RoBoT`

##### Must Use Optional Variables

‚Ä¢ `API_WEB_URL` : Ads Web Url eg: htts://github.com

‚Ä¢ `API_WEB_API` : Ads Web Api

‚Ä¢ `DONATION_LINK` : enter your gpay, pypal etc.. link

##### Optional Variables

‚Ä¢ `AUTH_USERS` : Username or ID of users to give access of inline search & pm filter. Separate multiple users by space. Leave it empty if you don't want to restrict bot usage.

‚Ä¢ `AUTH_GROUPS` : ID of groups which bot should work as autofilter, bot can only work in thease groups. If not given , bot can be used in any group.

‚Ä¢ `BOT_PICS` : Telegraph links of images to show in start message.( Multiple images can be used separated by space )

‚Ä¢ `SUPPORT_CHAT` : Your Support chat url or Your Link [with <code>https://</code> or <code>https://t.me/</code>] Don't use @

‚Ä¢ `STRING_SESSION` : User bot string session.

## Commands
```
‚Ä¢ /invite - to get the invite link of any chat which the bot is admin.
‚Ä¢ /logs - to get the rescent errors
‚Ä¢ /stats - to get status of files in db.
‚Ä¢ /connections - To see all connected groups
‚Ä¢ /settings - To open settings menu
‚Ä¢ /filter - add manual filters
‚Ä¢ /filters - view filters
‚Ä¢ /connect - connect to PM.
‚Ä¢ /disconnect - disconnect from PM
‚Ä¢ /del - delete a filter
‚Ä¢ /delall - delete all filters
‚Ä¢ /deleteall - delete all indexed files.
‚Ä¢ /delete - delete a specific file from index.
‚Ä¢ /info - get user info
‚Ä¢ /id - get tg ids.
‚Ä¢ /imdb - fetch info from imdb.
‚Ä¢ /search - To search from various sources
‚Ä¢ /start - To start the bot
‚Ä¢ /setskip - To skip number of messages when indexing files
‚Ä¢ /users - to get list of my users and ids.
‚Ä¢ /chats - to get list of the my chats and ids 
‚Ä¢ /leave  - to leave from a chat.
‚Ä¢ /disable  -  do disable a chat.
‚Ä¢ /enable - re-enable chat.
‚Ä¢ /ban  - to ban a user.
‚Ä¢ /unban  - to unban a user.
‚Ä¢ /channel - to get list of total connected channels
‚Ä¢ /broadcast - to broadcast a message to all users
‚Ä¢ /grp_broadcast - To broadcast a message to all connected groups.
‚Ä¢ /batch - to create link for multiple posts
‚Ä¢ /link - to create link for one post
‚Ä¢ /status - Your Heroku API Key to check dyno, bot uptime and bot working day prediction.
‚Ä¢ /set_template - To set a custom IMDb template for individual groups
‚Ä¢ /gfilter - To add global filters.
‚Ä¢ /gfilters - To view list of all global filters.
‚Ä¢ /delg - To delete a specific global filter.
```

## Variables

### Required Variables
* `BOT_TOKEN`: Create a bot using [@BotFather](https://telegram.dog/BotFather), and get the Telegram API token.
* `API_ID`: Get this value from [telegram.org](https://my.telegram.org/apps)
* `API_HASH`: Get this value from [telegram.org](https://my.telegram.org/apps)
* `CHANNELS`: Username or ID of channel or group. Separate multiple IDs by space
* `ADMINS`: Username or ID of Admin. Separate multiple Admins by space
* `DATABASE_URI`: [mongoDB](https://www.mongodb.com) URI. Get this value from [mongoDB](https://www.mongodb.com). For more help watch this [video](https://youtu.be/1G1XwEOnxxo)
* `DATABASE_NAME`: Name of the database in [mongoDB](https://www.mongodb.com). For more help watch this [video](https://youtu.be/1G1XwEOnxxo)
* `LOG_CHANNEL` : A channel to log the activities of bot. Make sure bot is an admin in the channel.
* `HEROKU_API_KEY`: Your Heroku API Key to check dyno, bot uptime and bot working day prediction
### Optional Variables
* `PICS`: Telegraph links of images to show in start message.( Multiple images can be used separated by space )
* `FILE_STORE_CHANNEL`: Channel from were file store links of posts should be made.Separate multiple IDs by space
* Check [info.py](https://github.com/MyBotzz/RashmikaBot/blob/master/info.py) for more optional variables
